# 图标生成说明

## 需要的图标尺寸

### PWA图标
- `icon-192.png` - 192x192像素
- `icon-512.png` - 512x512像素

### Android图标
- `mipmap-mdpi/icon.png` - 48x48
- `mipmap-hdpi/icon.png` - 72x72
- `mipmap-xhdpi/icon.png` - 96x96
- `mipmap-xxhdpi/icon.png` - 144x144
- `mipmap-xxxhdpi/icon.png` - 192x192

### iOS图标
- 需要多种尺寸（1024x1024, 180x180, 120x120等）

## 生成方法

### 方法1：在线工具
1. 访问 https://www.appicon.co/
2. 上传1024x1024的图标
3. 选择平台（iOS、Android、PWA）
4. 下载生成的图标包

### 方法2：使用ImageMagick
```bash
# 安装ImageMagick后
convert icon.png -resize 192x192 icon-192.png
convert icon.png -resize 512x512 icon-512.png
```

## 临时占位图标

在开发阶段，可以使用简单的占位图标。项目已包含基础的占位图标。
