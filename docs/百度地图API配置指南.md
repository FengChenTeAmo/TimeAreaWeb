# 百度地图API配置指南

## 错误码240：APP服务被禁用

如果遇到 `{status: 240, message: "APP 服务被禁用"}` 错误，请按以下步骤操作：

### 解决步骤

1. **登录百度地图开放平台**
   - 访问：https://lbsyun.baidu.com/
   - 使用你的账号登录

2. **进入控制台**
   - 点击右上角"控制台"
   - 选择"应用管理" -> "我的应用"

3. **找到你的应用**
   - 找到使用当前AK的应用
   - 点击"设置"或"编辑"

4. **启用地理编码服务**
   - 在"服务设置"或"功能服务"中
   - 找到"地理编码"服务
   - **勾选启用**该服务
   - 保存设置

5. **检查应用类型**
   - 确保应用类型选择正确
   - 如果是前端调用，选择"浏览器端"或"服务端"
   - 如果是后端调用，选择"服务端"

6. **检查IP白名单**
   - 在"访问控制"中
   - 如果是开发环境，可以设置IP白名单为 `0.0.0.0/0`（允许所有IP）
   - 如果是生产环境，建议配置具体的域名或IP

### 常见错误码说明

| 错误码 | 说明 | 解决方法 |
|--------|------|----------|
| 0 | 成功 | - |
| 1 | 服务器内部错误 | 稍后重试 |
| 2 | 请求参数非法 | 检查请求参数 |
| 3 | 权限校验失败 | 检查AK是否正确 |
| 4 | 配额校验失败 | 检查配额是否充足 |
| 5 | AK不存在或非法 | 检查AK是否正确 |
| 240 | APP服务被禁用 | **启用地理编码服务** |

### 验证配置

配置完成后，等待1-2分钟让配置生效，然后重新测试。

### 注意事项

1. 免费版百度地图API有调用次数限制
2. 需要实名认证才能使用
3. 生产环境建议配置域名白名单
4. 开发环境可以配置IP白名单为 `0.0.0.0/0`
